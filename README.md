# README
## Growing-cat(고양이 키우기)
pygame 라이브러리를 활용하여 고양이 육성 시뮬레이터를 만들었습니다.

## 육성 방법
아기 고양이로 시작해서 미니 게임을 통해 코인을 벌 수 있습니다.

그 코인을 이용해서 장난감, 밥을 구매 할 수 있습니다.

고양이의 스탯은 행복도, 피로도, 배고픔, 청결도가 있습니다.

행복도, 청결도가 0이 되거나, 피로도, 배고픔이 100이 되면 게임 오버입니다.

## 코인 버는 방법
하루가 지날때 코인을 획득합니다.

### 미니게임
미니게임을 통해서 코인을 획득할 수 있습니다.

* 미니게임 종류
    * 장애물 피하기 게임
    * 같은 그림 찾기

## 코인 사용
코인을 사용해서 스탯을 더 빠르게 증가시킬수 있는 유료 아이템을 구매 할 수 있습니다.

### 유료 아이템
* 낚싯대 장난감
* 실뭉치
* 고기
* 생선
* 츄르
